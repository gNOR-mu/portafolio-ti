Contexto de Formación

Este proyecto representa la culminación práctica de un programa intensivo de 480 horas enfocado en el ciclo completo de desarrollo de software.

Durante el proceso, se abordaron desde los fundamentos del diseño web (HTML5, CSS, JS, jQuery, Bootstrap) hasta el modelado avanzado de bases de datos relacionales, culminando con la programación de arquitecturas empresariales utilizando Java 21, JEE y el framework Spring Boot.

Seguridad

Hoy en día, la protección de datos es uno de los pilares más importantes de cualquier aplicación moderna, más aún en un E-commerce.

E-TravelGo integra e implementa Spring Security para una gestión robusta de autenticación y autorización. La aplicación integra Control de Acceso Basado en Roles (RBAC) y gestión segura de sesiones, garantizando la seguridad y protección de las rutas críticas del sistema contra vulnerabilidades comunes.

Patrón Strategy

Para el motor de precios y promociones del E-commerce, eludí la trampa común de anidar múltiples bloques if/else mediante la implementación del patrón de diseño Strategy.

Al definir la interfaz DiscountRule, el sistema puede evaluar dinámicamente una colección de estrategias de descuento inyectadas por Spring. Esto respeta el principio Open/Closed: permite agregar nuevas campañas promocionales en el futuro creando nuevas clases, sin necesidad de modificar ni romper el código base existente del carrito.
